## encrypt
some simple encrypt utils for go program.

一些简单实用的数据加解密编解码工具包

#### 安装
`go get -u github.com/gofuncchan/encrypt`

#### 加解密工具包：

- 哈希 
    - md5
    - sha2
- 对称加解密
    - des
    - tdes
    - aes
- 非对称加解密
    - rsa
    - ecies
- 数字签名
    - ecdsa

#### 数据编解码包

- base64
- url解析
- 十六进制编码
- json编解码
- yaml编解码
- cvs编解码
